Getting Started

Follow these simple steps to set up Syft Learning for your family

1

Install the app on the parent's phone

2

Login with the parent's account

3

Create the bots

Go to the bot editor and create a few bots.

4

Set a pin

Tap on the gear icon in the top right corner and choose Set Pin.

5

Add a profile for each teen

6

Turn on notifications

7

Install the app on the teen's phone

8

Login with the parent's account on teen's device

9

Select the teen's profile

10

Teen - Select a bot and start chatting
